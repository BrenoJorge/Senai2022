Laços de repetição

- Do While (Faça enquanto)
	Ex:
	do{
		int sair = read.nextInt();
	}while(sair == 0);
	
- While (Enquanto)
	Ex:
	int sair = 0;
	while(sair == 0){
		sair = read.nextInt();
	}
- For (Para)
	Ex:
	for (int i = 0; i < 10; i++){
		System.out.println("vezes = "+i);
	}
